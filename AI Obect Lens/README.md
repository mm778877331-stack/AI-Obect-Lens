 Advanced system  For object recognition using TensorFlow and Python 3.12, with real-time neural weight extraction and deep machine learning analysis.

## 🚀 Project Concept

AI Object Lens is a computer vision tool designed to bridge the gap between complex deep learning models and user-friendly interfaces. The system uses a pre-trained MobileNetV2 - VGG16 - ResNet50 architecture to identify objects in uploaded images and provides a transparent view of the neural network s "brain" by displaying weights and biases.

| Domain | Technologies |

| Language | Python 3.12 |

## ✨ Features
- High-Speed ​​Recognition: Uses MobileNetV2 for fast and accurate object classification.
- Neural Vision: Displays real-time weight and bias samples for academic analysis.
- Professional Dashboard: Future-proof design with custom CSS for a premium viewing experience.
- Model Summary: Provides a detailed analysis of 155 layers within the neural network.
- Scalability: Ready to export to TFLite for integration with Flutter applications.



## 📊 Model Details

| Element | Details |
|--------|----------|
| Architecture: | MobileNetV2, VGG16, ResNet50 |
| Dataset: |  ImageNet (1.2 million images) |
| Input Size | 224×224 pixels |
| Optimization | Enable custom oneDNN processes |
