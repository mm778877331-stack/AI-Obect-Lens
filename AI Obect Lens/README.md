# AI Object Lens 🔍
An advanced Image Recognition System built with TensorFlow and Python 3.12, 
featuring real-time neural weight extraction and deep learning analysis.

## 🚀 Project Idea
The AI Object Lens is a computer vision tool designed to bridge the gap between 
complex Deep Learning models and user-friendly interfaces. It uses a pre-trained 
MobileNetV2 - VGG16 - ResNet50 architecture to identify objects in uploaded images 
and provides a transparent look into the neural network s "brain" by displaying
weights and biases.

## 🛠 Tech Stack
Language: Python 3.12
Deep Learning Framework: TensorFlow / Keras
Environment: Jupyter Notebook / VS Code
GUI: IPyWidgets & HTML/CSS Styling
Image Processing: Pillow (PIL) & NumPy

## ✨ Features
High-Speed Recognition: Uses MobileNetV2 for fast and accurate object classification.
Neural Insights: Displays real-time Weights and Bias samples for academic analysis.
Professional Dashboard: A futuristic UI/UX designed with custom CSS for a better presentation experience.
Model Summary: Provides a detailed breakdown of the 155 layers within the neural network.
Scalability: Ready to be exported as TFLite for Flutter mobile integration.

## 📊 Model Details
Architecture: MobileNetV2 - VGG16 - ResNet50
Dataset: ImageNet (1.2 Million images)
Input Size: 224x224 pixels
Optimization: oneDNN Custom Operations enabled
